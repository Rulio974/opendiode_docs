# Installation virtualisée

# Pré-requis

Installation non recommandée

Cette installation n'est actuellement pas recommandée. Nous travaillons sur une version virtualisée simplifiée.

Elle permet néanmoins d'accéder à des serveurs web utilisés dans le cadre d'une démonstration.

Temps d'installation : 15 minutes hors téléchargement

La machine hôte devra au minimum posséder :

  • Mémoire vive (RAM) : 8GB.
  • Processeur (CPU) : N'importe quel modèle AMD ou Intel 64-bit architecture amd64 (2 cœurs physiques ou plus recommandés).
  • Stockage : 60GB minimum.

Vous devrez également disposer de la dernière version de VirtualBox disponible ici :

  • Windows

  • Linux
